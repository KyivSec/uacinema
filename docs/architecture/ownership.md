# Component ownership

| **frontend-web** | Маргарита Киян            | (буде уточнено)           | Відповідає за структуру та реалізацію React SPA, маршрути, сторінки каталогу й деталей українських фільмів, інтеграцію з REST-API бекенду та узгоджений інтерфейс користувача. | Тег: `component:frontend-web`, епік на борді задач для фронтенду. |
| **backend-api**  | Андрій Солдатов           | (буде уточнено)           | Відповідає за REST-ендпоінти `/api/...`, бізнес-логіку відбору українських фільмів, обробку помилок, а також інтеграцію з БД та TMDB API. | Тег: `component:backend-api`, епік для серверної частини. |

| **tmdb-gateway** | Андрій Солдатов           | (можливі додаткові dev’и) | Відповідає за модуль інтеграції з TMDB API: формування запитів, фільтрацію під українські фільми, мапінг відповіді та кешування. | Тег: `component:tmdb-gateway`, епік «TMDB integration». |
| **observability-ci** (майбутній) | Андрій Солдатов або окремий DevOps (за домовленістю) | (буде уточнено) | Відповідає за налаштування CI/CD, логування бекенду та базовий моніторинг (коли компонент буде впроваджуватися). | Тег: `component:observability-ci`, епік «CI/CD & monitoring». |
